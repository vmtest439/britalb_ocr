# https://t.me/ru_vm (BritLab)

### Google Gemini OCR (не работает с RU IP)
1. Add your API key to `gemini_recognizer.py`
2. Install dependencies:
```bash
pip install requests
```
3. Run the script:
```bash
python3 gemini_recognizer.py
```
